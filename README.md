# Spring-Android-PhoneApp
스프링서버와 안드로이드를 연동해서 전화번호부를 만들었다.

## 🏅기술
##### Spring JPA
##### Android Retrofit, RecycleView
##### h2 DB
<br/>

## 🏅기능
##### 연락처 전체보기, 연락처 등록하기, 연락처 수정하기, 연락처 삭제하기 
##### 모두 DB와 연동
<br/>
